# 💬 Kanmani AI

**Kanmani AI** is a sleek, real-time conversational web application built using **React.js** and powered by the **Gemini API**. It provides intelligent, human-like interaction through a modern and responsive chat interface.

## 🚀 Project Overview

This project was created as part of a personal initiative to explore conversational AI and frontend integration. The app allows users to interact with an AI bot that responds using Gemini's advanced language model.

## ✨ Features

- 🔥 Real-time messaging experience
- 🤖 Gemini API integration for natural language understanding
- 🧠 Context-aware responses
- 💻 Built with React for dynamic UI updates
- 🎨 Clean and responsive user interface

## 🛠️ Tech Stack

| Technology   | Purpose                             |
|--------------|--------------------------------------|
| React.js     | Frontend framework                   |
| Gemini API   | AI-powered natural language response |
| Vite         | Fast React development environment   |
| CSS          | Styling the interface                |
